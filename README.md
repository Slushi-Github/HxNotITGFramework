# HxNotITG Framework

This is the library with the necessary code to create [NotITG](https://www.noti.tg/) modcharts using Haxe with [HxNotITG](https://github.com/Slushi-Github/HxNotITG).

**Don't try to use this library in a standard Haxe project; it's designed to work with [Reflaxe/Lua](https://github.com/Davvex87/reflaxe.lua)!**

See the [TODO list](./TODO.md) for future features.

## Installation

You can install HxNotITG Framework using:

Haxelib (Not available yet):

```bash
haxelib install hxnotitgframework
```

Or GitHub:

```bash
haxelib git hxnotitgframework https://github.com/Slushi-Github/HxNotITGFramework.git
```
